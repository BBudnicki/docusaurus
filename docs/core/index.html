Core Data Model

Core Data Model (CDM) is a data model that glues Projects, Datasets, and Observations together. As a central object, it is required for complete implement of this standard.

caution

This is the extent of the CDM Core schema. Additional attributes required in the programOrCampaign entity for any given implementation should be considered as extensions to this schema.
